## 🐛 常见问题

### 登录失败
- 检查回调地址是否正确
- 确认域名使用 HTTPS（生产环境）
- 验证应用 ID 和域名

### 回调错误
- 检查回调 URL 格式
- 确认路由配置正确
- 验证 CORS 设置

### 组件加载失败
- 检查依赖包是否正确安装
- 确认导入语句正确
- 验证 TypeScript 类型
