🐛 常见问题排查

1. 验证码按钮不显示

可能原因： - CSS样式问题 - JavaScript错误 - 组件渲染失败 - 路由配置错误 检查方法： 1. 打开浏览器开发者工具 2. 查看Console标签页是否有错误 3. 查看Elements标签页检查按钮元素是否存在 4. 检查Network标签页是否有资源加载失败

2. 点击发送按钮无反应

可能原因： - JavaScript函数未绑定 - 事件处理函数错误 - API调用失败 - 网络连接问题 检查方法： 1. 查看Console标签页的错误信息 2. 检查Network标签页的API请求 3. 确认手机号/邮箱格式是否正确 4. 检查倒计时功能是否正常

3. redirect_uri_mismatch 错误

解决方案： 1. 运行 ./fix-authing-config.sh 脚本 2. 在Authing控制台添加正确的回调URL 3. 确保端口号匹配 4. 检查redirectUri配置

🔧 快速修复脚本

如果遇到问题，可以运行以下脚本：

# 检查Authing配置 ./fix-authing-config.sh # 重启开发服务器 npm run dev # 检查端口占用 lsof -i :3000 -i :3001 -i :3002

📞 测试步骤

  1. 确保开发服务器在端口3000或3001运行
  2. 点击上面的页面链接
  3. 检查验证码按钮是否显示
  4. 输入手机号/邮箱，点击发送按钮
  5. 检查倒计时是否正常
  6. 查看浏览器控制台是否有错误
